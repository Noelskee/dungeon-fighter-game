# Dungeon Fighter Game

This is the README for the Dungeon Fighter Game project. 

## Features
- Fast-paced gameplay
- Multiple dungeons to explore
- Various character classes

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Noelskee/dungeon-fighter-game.git
   ```
2. Navigate to the project directory:
   ```bash
   cd dungeon-fighter-game
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
Run the game using:
```bash
npm start
```
